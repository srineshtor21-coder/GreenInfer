Features

Built for efficiency

Every layer of GreenInfer is designed to reduce unnecessary compute while preserving answer quality.

🔬
Complexity Scoring
DistilBERT classifier trained on 600 labeled examples rates every prompt 0–100. 98.9% validation accuracy across 4 tiers.
⚡
T5 Prompt Optimizer
Silently rewrites prompts to remove filler words before inference — cutting input tokens by an average of 35%.
🌍
Carbon-Aware Routing
Uses hourly ERCOT grid intensity estimates to defer expensive queries when the grid is running dirty.
🪜
Cascade Engine
Starts with the smallest model. Only escalates to medium or large if confidence is below threshold — inspired by FrugalGPT.
💡
Smart Preview Mode
For complex queries, shows a 2-sentence summary + outline first. User confirms before the full expensive response runs — preventing wasted reruns.
📊
Per-Response Metrics
Every answer shows energy used (mWh), CO₂ emitted, tokens saved, and a Green Efficiency Score 0–100 with an improvement tip.

Python Framework

Drop-in green inference

Install the package and get energy-aware responses in three lines of code.

Python 
from greeninfer import GreenInfer

# Automatically routes to the right model tier
gi = GreenInfer()
result = gi.chat("Explain photosynthesis")

# Full energy transparency
print(result.response)
# → "Photosynthesis is the process by which..."

print(result.energy_mwh)
# → 0.9  (small model, 98% less than large)

print(result.model_tier)
# → "small"

print(result.co2_grams)
# → 0.000178
